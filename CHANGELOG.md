# Changelog

All notable changes to `laravel-content-summarize` will be documented in this file.

## v1.0.2 - 2026-02-21

Security: bumped minimum Laravel version to ^10.48.29 to address file validation bypass vulnerability (CVE). Dropped Laravel 9 support (EOL since Feb 2024).

## 1.0.0 - 2024-06-06

- Initial release
- Support for AI-powered text summarization
- Customizable summary length and language options
- Comprehensive documentation and examples
